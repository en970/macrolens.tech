# MacroLens - AI Powered Nutrition

Stop guessing. Start snapping. Your nutrition, automated by AI.

## Features
- **Identify Meals**: Instantly analyze calories and macros with GPT-4 Vision.
- **AI Nutritionist**: Expert insights and tailored advice in your pocket, 24/7.
- **Barcode Scanner**: Quick scan for instant facts on packaged foods.
- **Dynamic Growth**: Real-time sign-up counter for waitlist.
- **Mobile First**: Fully responsive design for iOS and Android web views.

## Tech Stack
- Next.js 15
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide Icons

## Deployment
This project is configured for static export and hosted on GitHub Pages. Deployment is handled automatically via GitHub Actions.
